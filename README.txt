Interesting Files Module
Sleuth Kit Framework C++ Module
May 2012


This module is for the C++ Sleuth Kit Framework.


DESCRIPTION

This module is a reporting module that looks for files matching
criteria specified in an input file. The module posts its findings 
to the blackboard.

USAGE

Configure the reporting pipeline to include this module and to
pass the path of an input file as a module arguments string. A 
sample input file is provided (interesting_files.xml).
